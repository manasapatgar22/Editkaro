A polished single-page portfolio demonstrating video-editing case studies,
filters, theme toggle, contact drawer and modal video player. Built with plain
HTML/CSS/JS and GSAP for animations.
### Features
- Filterable video grid
- Search
- Lightbox modal with YouTube embed
- Dark / Light theme with LocalStorage
- Preloader
- Contact drawer (EmailJS integration placeholder)
- Animated stats
- GSAP entrance animations
### Run locally
1. Clone the project folder
2. `cd editkaro-portfolio`
3. Open `index.html` in browser (or serve with a static server)
### Deploy
- Deploy on Vercel / Netlify / GitHub Pages by connecting the repository.
